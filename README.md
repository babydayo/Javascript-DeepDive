## 📚 스터디 진행방식

- 룰렛 돌려서 발표자 2명 선정, 진행
- 3장 발표하면, 50%씩 나눠서 발표 진행
- 발표자가 먼저 공유할 내용(질문) 발표

- 정적이 흐르면 발표자가 토론 유도
